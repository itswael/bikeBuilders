# Assets Folder

This folder contains application assets like icons and splash screens.

## Required Assets

Place the following files in this directory:

- `icon.png` - App icon (1024x1024 px)
- `splash.png` - Splash screen image (1284x2778 px)
- `adaptive-icon.png` - Adaptive icon for Android (1024x1024 px)

You can generate these assets using online tools or design software.

For development, you can use placeholder images or Expo's default assets.
